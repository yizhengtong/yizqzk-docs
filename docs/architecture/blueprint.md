## A. 境界跨度

### 境界分级

```
筑命 → 谌我 → 揖别 → 证我
```

### 核心机制

- **顺序性里程碑记录**：每个境界的条件只需达成过一次即永久记录，无需同时满足
- **顺序锁定**：未突破当前境界前，下一境界的条件不予检测/记录
- **属性叠加**：境界属性层层叠加，突破至新境界后保留所有先前属性

### 突破条件

| 境界 | 条件 |
|------|------|
| **筑命** | 经验值 20 级，道宫×1，总天赋 ≥ 6 |
| **谌我** | 经验值 30 级，道宫×1，道宫规模 ≥ 576 方块，总天赋 ≥ 12，史诗天赋 ≥ 1 |
| **揖别** | 道宫 ≥ 2，道宫总规模 ≥ 32,000 方块，总天赋 ≥ 24，传说天赋 ≥ 2 |
| **证我** | 道宫 ≥ 3，每座道宫规模 ≥ 64,000 方块，总天赋 ≥ 64，神话天赋 ≥ 6，传说天赋 ≥ 12 |

> 道宫规模 = 所有道宫已建造方块数之和

## C. 道宫

### 落点

- 初始拥有 1 座落点
- 首次击杀末影龙 / 凋零各一次，可解锁 1 次新落点

### 可建筑范围

- 激活条件：落点中心 R=10 范围内可打开道宫面板
- 初始边长：L₀ = 20 + 2×10 = 40（正方体边长）
- 单次扩展成本：`[ (当前边长+2)³ - 当前边长³ ] / (10 × 落点数量²)`
- 建筑形状：固定正方体

### 影响力范围

- **落点影响范围边长**：`影响边长 = 当前可建筑边长 × (4 ^ 落点总数)`
- **终端衰减增益**：`终端增益 = 基础增益 × (已建造方块数 / 可建筑方块总数)`

## D. 天赋 ✅ 已实现

### 概述

- **统称**：天赋
- **种子类型**：随影（实体）/ 词缀（物品）
- **等级**：无硬性限制
- **稀有度**：神话 → 传说 → 史诗 → 精英 → 平凡
- **绑定方式**：实体绑定（默认持久化）/ 物品绑定（5 槽位）

### 天赋五性

| 天性 | 英文 | 定位 |
|------|------|------|
| 残响 | ECHO | 攻击类 |
| 铭刻 | INSCRIPTION | 回复类 |
| 显化 | MANIFESTATION | 机制类 |
| 本形 | ORIGIN | 防御类 |
| 升灵 | ASCENSION | 被动类 |
